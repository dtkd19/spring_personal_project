## 스프링 게시판 프로젝트
- Spring을 사용해서 기본적인 커뮤니티 게시판을 구현한 프로젝트입니다.

## ❓ 프로젝트 소개
Spring과 DB를 사용하여 MVC 패턴의 이해도를 높히고 데이터의 저장 / 조회 / 수정 등의 작업을 익힐 수 있도록  
게시판을 만들어보았습니다.
